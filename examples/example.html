jQuery Radio Toggle Examples

Default behavior

The script transforms radio buttons into toggle buttons.

Specifying a callback

To add a callback on button click, pass in a function to the buttonClick option during initialization.

Specifying a theme

To specify a theme, pass the 'theme' option during initialization. The theme will be added as a class on the container. To create your own theme, just follow/modify the included CSS file. The following classes and styles are applied:

  • { display: none } to all input elements
  • Class [containerClass] to the containers
  • Class [theme] to the containers ([containerClass].[theme])
  • Class [activeClass] to the active button
  • Class [inactiveClass] to the inactive buttons
  • Class [disabledClass] to the disabled buttons
  • Class 'first' to the first button
  • Class 'last' to the last button
